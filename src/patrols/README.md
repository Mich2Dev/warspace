# Patrullas (`src/patrols`)

Trenes de **3 naves en fila india** usando los diseños **E3 → E2 → E1**.

| Archivo | Rol |
|---------|-----|
| `PatrolSquadManager.js` | Spawn, rutas, formación, paralizador del comandante |
| `squadRoles.js` | Stats por rol, `COLUMN_FORMATION`, minimapa |
| `data/patrol_squads.json` | 3 rutas estratégicas (Alpha / Este / Oeste) |

Consola al cargar: `[PatrolSquad] 3 formación(es) en ruta`
